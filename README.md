# Personal Portfolio Website

This is my personal portfolio website built with HTML, CSS and JavaScript. It shows my work, skills and contact details in a clean layout. The site is fully responsive and works well on all devices.

## Features

- Home section with intro
- About section with skills and background
- Projects section with demos and code links
- Contact section with form or direct links
- Responsive design
- Smooth animations

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone or download the repository  
   `git clone <your-repo-link>`
2. Open the folder.
3. Open `index.html` in any browser.
